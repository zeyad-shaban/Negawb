Frequently Asked Questions

  • Fix Problem

    • Notifications not working

    Go to Your profile and click on 'Subscribe to push messaging', you will get a window pop up asking you to allow notifications, click allow

    can't alow notifications

    This means that your browser have blocked notifications, or you're on Incognito window

    for desktops, beside the url on left you will find a bunch of icons ( as shown in image below ), click on them. you should see that notifications are blocked, click on the 'x', now when you subscribe to notifications you should not get any errors


    For phones,

    Can't see posts in homepage

    You should find message saying 'Your Distraction Free settings doesn't allow posts to show in homepage' so don't worry this is not a problem, this is the one of Dfreemedia unique behaviours, to cancel it go to your profile and click up Distraction Free, now click on 'posts to show in homepage' and choose your favourite category

    Page not loading properly

    meaning that your page is not loading javascript, again check for permissions you gave to website and make sure to allow javascript, if you don't know how check the above 'can't allow notifications'

    Server error 500

    This error is caused due to an error from the website developers, if you ever faced this issue Report problem here and paste the url and we will fix it as soon as possible, meanwhile you can try to login, maybe this will fix the issue

    Server error 404

    You entered a valid url, if you got this error after clicking on a link or button from our website then please Report problem and provide us with the element you clicked

  • Manage your account

    • Is my password secure?

    Yes, as they are turned into hashed passwords first, which make it almost impossible for us and any hacker to crack your password, so ya don't worry buddy ;-)

    Protect profile picture

    Go to your Profile and click on 'privacy', head to 'Who can see profile image', you can either make it every one, friends only, or no one, tho friends only is still beta

    Stop getting friend and group requests

    Again click privacy in your profile, to block friend requests uncheck 'allow friend requests' (the last choice)
    to stop getting group requests, click 'Who cann add me to groups', you can either choose everyone, friends only, no one.

    How to reset password

    For now you can't, the button is there and everything exists, but we are still working very hard to add email vertifications, if you want to help us the simplest way is to share the app with your friends :D

  • Get the most out of dfreemedia

    • Dfreemedia was made to be as less distractive as possible, so if you want to get the most of it you should consider minimizing posts you see, minimizing messages, groups, etc... please explore the Distraction Free menu in you profile

    Send important message

    one of the unique features Dfreemedia provides, ability to mark messages as important and others as normal, simply check the checkbox beside the send button and tada your message is important. You can only send 3 important messages each day for each chat room

    Block specific notification

    Yes you heard it right, in Dfreemedia you can have the full control of notification types and which ones to receive, go to your profile(again) choose Distraction Free

    pro tip, you can block Normal messages notifications and allow important ones

    Turn Dfreemedia to chat app

    Go to profile and open Distraction Free, now check the last option 'Chat only mode'

    limit posts

    We already have a system that doesn't allow users to mkae more than 3 posts/day (with the exeption of Fun Area), Go to your profile and choose 'Distraction Free, block all categories you don't like to get posts from (last option), you can also 'hide Followers/Friends posts in homepage', you can specify which category to be your displayed homepage, choose none if you don't like posts to show there
    please explore 'Distraction Free' settings because that's where we shine, where you have all the power (which none of the other social media platforms provide)

    Hide comments

    Go to your profile and check 'Hide comments' (second option)
